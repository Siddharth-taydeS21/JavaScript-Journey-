● location :
Location Represents the wed address of the project or a window or a website.
location.reload()- which reloads the current page. 

● history :
it represents the tabs history of the window object.
it has the history.back() method to go to the  previous tab & history.forward() method to go to the next tab.
browsers back (<-) and forward (->) buttons uses this functionalities.

it has history.go() method to go to the particular tab we want. like this :-
history.go(-1) to go to the previous tab;
history.go(2) to go to the next 2end tab;
history.go(-2) to go to the previous 2end tab;

this method is used to switch pages according to history.length

● innerHeight :
window.innerHeight represents the height of the tab in browser;
window.outerHeight represents the height of the actual screen of our device;

● innerWidth : 
window.innerWidth & window.outerWidth work the same about width of the window and screen

● open() method gives you the window of the current tab and;
and when you give it url as an argument it can open that link in new tab;

● close() method use to close the tabs, but you can only close that tabs using close() method which has opened with JS or window.open() method.
