Chrome dev tools are helpful for these - 

1) console - logs, warnings, tables, groups, custom styles using (%c with string or anything we want)

2) Sources tab - 
explore file system of the project, 
use breakpoints and debuggers,
edit javascript in code editor tab;
know about scope, functions, call stack in debug tab.

3) snippets - save repetitive code in snippets and reuse when you want.

4) Styling with Inspect elements - 
inspect html elements to change text or content,
copy CSS path of a particular element to write his style,
copy JS path of a particular element to write is as a variable.
color picker,
color magnifier,
shadow customization tab, 
get hex values, rgb values & etc.
